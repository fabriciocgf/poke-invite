# 🎮 PokeInvite - Dynamic Pokémon Party Invitation

A full-stack, containerized web application built with Python and Flask that generates a highly customizable, dynamic Pokémon-themed event invitation. Every time a guest visits the page, they are assigned a random Pokémon, and the entire UI (backgrounds, text colors, and shadows) mathematically adapts to perfectly match that Pokémon's predominant color while maintaining accessible text contrast.

It includes a built-in RSVP system that saves guest data locally and automatically sends a themed HTML confirmation email.

## ✨ Features
* **Dynamic Theming:** Extracts the predominant Hex color from the Pokémon's image and perfectly themes the page.
* **Smart Contrast:** Uses Relative Luminance calculations to automatically switch text between light and dark to ensure readability.
* **Light/Dark Mode:** Adapts to the user's OS preference by default, saves manual toggles to `localStorage`, and updates seamlessly via CSS variables.
* **Interactive Options:** Guests can manually choose their favorite Pokémon or hit "Re-roll" to get a new random theme.
* **Smart RSVP System:** Saves guest names, emails, assigned Pokémon, and theme colors to a local JSON file. Includes server-side duplicate email prevention with beautifully animated, baked-in UI alerts (no clunky browser popups!).
* **Secure Admin Dashboard:** A password-protected portal (`/admin`) to view your headcount, delete duplicate RSVPs, and download your guest list as a CSV. Secured with hashed passwords, HTTPOnly cookies, and cryptographically signed time-bound sessions.
* **Add to Calendar Integration:** Automatically generates Google Calendar links for guests upon successful RSVP, pre-filled with the event's title, date, times, and location.
* **Full Localization (i18n):** All frontend UI text and email labels are extracted into a `lang.json` file. Easily translate the entire app to any language without touching a single line of code.
* **Automated Emails:** Sends a beautifully styled, dynamic HTML confirmation email (in your chosen language) using SMTP.
* **Zero-Downtime Config:** Event details and language translations are loaded dynamically from `config.json` and `lang.json` so you can update them live in production without restarting the server.
* **Blazing Fast CDN Optimization:** Converts heavy media to compressed `.webp` formats and uses aggressive `Cache-Control` headers so assets load instantaneously through Cloudflare.
* **Production Ready:** Fully containerized with Docker and Gunicorn, designed to sit seamlessly behind a Cloudflare Zero Trust Tunnel.

## ⚙️ Configuration

### 1. Environment Variables & Security (`.env`)
Create a `.env` file in the root directory to store your credentials and security keys. 
*To generate your password hash, run `python -c "from werkzeug.security import generate_password_hash; print(generate_password_hash('your_password'))"`*

**⚠️ Docker Warning:** You MUST double every dollar sign (`$$`) in your generated password hash, otherwise Docker will mistake them for system variables and corrupt your password.

\`\`\`env
# Email Configuration (Use a 16-letter App Password if using Gmail)
SMTP_EMAIL=your_email@gmail.com
SMTP_PASSWORD=your_16_letter_app_password

# Admin Dashboard Security
FLASK_SECRET_KEY=super-secret-random-key
ADMIN_PASSWORD_HASH=scrypt:32768:8:1$$yourSalt$$yourHash
\`\`\`

### 2. Event Details (`config.json`)
Edit this file to customize your party details. You can update this file while the server is running, and the changes will reflect instantly on the next page load. Note the ISO formatted strings (UTC) for Google Calendar generation.
\`\`\`json
{
    "title": "Fabrício's Party!",
    "message_template": "Be like {name} and catch the fun at my birthday!",
    "date": "Saturday, Oct 12th",
    "location": "Pallet Town (My House)",
    "time": "14:00 - Gotta catch 'em all!",
    "theme": "auto", 
    "show_theme_button": true,
    "email_subject": "You are officially invited! 🎉",
    "email_message": "Thanks for RSVPing, {guest_name}! We can't wait to see you there.",
    "calendar_start": "20261012T170000Z",
    "calendar_end": "20261012T220000Z"
}
\`\`\`

### 3. Translation (`lang.json`)
All UI text is isolated in `lang.json`. To translate the app to a new language, simply edit the values in this file. Changes map instantly to the live Docker container.

## 🛠️ Data Pipeline (Updating the Pokémon List)

If you ever want to add new Pokémon or regenerate the data, use the included pipeline scripts.

**1. Download the Images:**
This script downloads raw `.png` files directly from the source into the `pokemon_logos/` folder.
\`\`\`bash
python download_images.py
\`\`\`

**2. Compress the Images:**
This script converts the heavy `.png` images into highly optimized `.webp` formats (reducing file size by ~80%) for instantaneous CDN streaming, then deletes the old PNGs.
\`\`\`bash
python compress_images.py
\`\`\`

**3. Process Colors & Names:**
This script resizes the images, filters out dark outlines/transparent pixels, calculates the most vibrant predominant color, and pings the PokéAPI to get the correct character name. It outputs the final `refined_colors.json`.
\`\`\`bash
python process_colors.py
\`\`\`

## 🚀 Running Locally (Development)

It is recommended to use `uv` for blazing-fast dependency management.

1. **Create and activate a virtual environment:**
   \`\`\`bash
   uv venv
   # On Windows:
   .venv\Scripts\activate
   # On Mac/Linux:
   source .venv/bin/activate
   \`\`\`
2. **Install dependencies:**
   \`\`\`bash
   uv pip install -r requirements.txt
   uv pip install python-dotenv  # Required for local .env loading
   \`\`\`
3. **Run the Flask server:**
   \`\`\`bash
   python app.py
   \`\`\`
4. Open your browser to `http://localhost:5000`.

## 🐳 Running in Production (Docker)

This app is optimized to run via Docker using `gunicorn`. The `docker-compose.yml` file is configured to mount the `guests/` folder, `config.json`, and `lang.json` directly to your host machine so data is never lost and configurations can be updated live.

1. **Build and start the container:**
   \`\`\`bash
   docker compose up -d --build
   \`\`\`
2. **Check the logs (optional):**
   \`\`\`bash
   docker compose logs -f
   \`\`\`
3. **Stop the container:**
   \`\`\`bash
   docker compose down
   \`\`\`

### Cloudflare Zero Trust Routing
To expose this securely to the internet without opening router ports:
1. Run your Cloudflare Tunnel on the host machine.
2. In the Cloudflare Zero Trust Dashboard, create a Public Hostname pointing to the Docker container.
3. Set the Service Type to `HTTP` and the URL to `localhost:5000`.
